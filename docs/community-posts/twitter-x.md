**Tweet 2:**
> Two install modes:
>
> 🌐 Remote — runs on your Mac/PC, connects to Batocera over SSH
> 🎮 Native — runs directly on the Batocera device, auto-starts
>
> One installer figures out which:
> `./install.sh`
>
> (Windows: install.bat detects WSL or Git Bash)

**Tweet 3:**
> What you get:
> 📊 Live CPU/RAM/temperature
> 🎮 ROM library with cover art
> 📁 File manager for /userdata
> 🖥️ Terminal (with command-safety guards)
> 📜 Live log viewer
> ⚙️ Per-system batocera.conf editor
> 📱 Mobile responsive

**Tweet 4:**
> Tech: Python/Flask + vanilla JS + NES.css. 104 automated tests (82 API + 22 Playwright) running on GitHub Actions.
>
> ⚠️ For home network use only — it's basically root access to your Batocera. Don't expose the port to the internet.
