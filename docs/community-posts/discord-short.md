Hey folks 👋 just released **Batocera Web Dashboard PRO v2.0** — a retro-styled web UI you run alongside Batocera to manage ROMs, watch live stats, browse files, and run commands, all from any browser on your network.

Two modes: **Remote** (runs on your PC, connects via SSH) or **Native** (runs on the Batocera device, auto-starts).

✅ Live CPU/RAM/temp graphs
✅ ROM library with cover art
✅ File manager + terminal
✅ Mobile-friendly
✅ Open source (MIT)
✅ 100+ automated tests

Install:
```
git clone https://github.com/DavidSchuchert/Batocera-WebDashboard-Pro.git
cd Batocera-WebDashboard-Pro && ./install.sh
```
