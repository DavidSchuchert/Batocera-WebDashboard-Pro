# Community post drafts

Copy-paste templates for announcing the project on different platforms.
Each is tuned to the platform's culture and length expectations.

- `forum-batocera.md` — long-form post for [forum.batocera.org](https://forum.batocera.org)
- `reddit-batocera.md` — for [r/batocera](https://reddit.com/r/batocera) and [r/emulation](https://reddit.com/r/emulation)
- `discord-short.md` — short version for Discord servers (Batocera, RetroArch, etc.)
- `twitter-x.md` — single tweet + thread version

## Tips

- Post during European/US evening hours for best engagement
- Reply to comments quickly (especially in the first 2 hours)
- Don't cross-post the *exact same text* — Reddit and forums frown on this
- Always link to a screenshot or the demo GIF; visual = engagement
